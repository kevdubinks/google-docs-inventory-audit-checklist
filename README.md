# Google Docs Inventory Audit Checklist

A Google Docs template with the following structure:\n\n**Title:** Inventory Audit Checklist\n\n**Columns:**\n- Date\n- Product Name\n- Current Quantity\n- Target Quantity\n- Alert (if quantity is below target)\n\n**Example Data:**\n| Date       | Product Name  | Current Quantity | Target Quantity | Alert |\n|------------|--------------|------------------|----------------|-------|\n| 01/01/2023 | Product A     | 5                | 10            | Yes   |\n| 01/08/2023 | Product B     | 7                | 5            | No    |\n\n**Formulas:**\n- Use conditional formatting to highlight cells with alerts.\n- Use the `=IF` function to compare current and target quantities.\n\n**Usage Instructions:**\n1. Open the Google Docs template.\n2. Replace example data with your inventory information.\n3. Update the target quantities as needed.
